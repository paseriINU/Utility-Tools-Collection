# サンプルツール・テンプレート集

このフォルダには、サンプルツールやテンプレートを配置します。

## 提案ツール一覧

以下は今後作成を検討しているツールの一覧です。

### PowerShell/Batch

| ツール名 | 説明 | 優先度 |
|----------|------|--------|
| log-collector | 複数サーバからログファイルを一括収集 | 高 |
| service-monitor | Windowsサービスの状態監視と自動再起動 | 中 |
| scheduled-backup | フォルダの定期バックアップ（世代管理付き） | 中 |
| disk-usage-reporter | ディスク使用量レポート生成 | 低 |

### Shell (Linux/Unix)

| ツール名 | 説明 | 優先度 |
|----------|------|--------|
| process-monitor | プロセス監視と異常時通知 | 高 |
| db-backup | データベースバックアップスクリプト | 中 |
| deploy-script | アプリケーションデプロイ自動化 | 中 |
| log-rotate | カスタムログローテーション | 低 |

### VBA (Excel/Word)

| ツール名 | 説明 | 優先度 |
|----------|------|--------|
| csv-bulk-import | 複数CSVの一括インポート・整形 | 高 |
| pdf-batch-convert | Excel→PDF一括変換 | 中 |
| data-validator | データ検証・エラーチェック | 中 |
| report-generator | 定型レポート自動生成 | 低 |

### JavaScript (ブラウザ/ブックマークレット)

| ツール名 | 説明 | 優先度 |
|----------|------|--------|
| form-auto-fill | フォーム自動入力 | 高 |
| json-formatter | JSON整形・表示 | 中 |
| regex-tester | 正規表現テスター | 中 |
| table-exporter | HTMLテーブル→CSV変換 | 低 |

### Python

| ツール名 | 説明 | 優先度 |
|----------|------|--------|
| file-organizer | ファイル自動整理（日付・種類別） | 高 |
| encoding-converter | 文字コード一括変換 | 中 |
| report-aggregator | 複数ファイルの集計レポート | 中 |
| api-client-template | REST APIクライアントテンプレート | 低 |
