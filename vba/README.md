# VBA Tools

Excel VBAマクロによる自動化ツール集です。

## 📂 サブフォルダ

### SQL_生成ツール/
Oracle用SELECT文を対話的に生成するツールです。

- Excel上でGUIを使ってSQL文を組み立て
- JOIN（INNER, LEFT, RIGHT, FULL OUTER, CROSS）対応
- サブクエリ、WITH句（CTE）、UNION対応
- 集計関数（COUNT, SUM, AVG, MAX, MIN）対応
- GROUP BY / HAVING / ORDER BY対応
- SQLヘルプシートで初心者も安心
- 生成履歴の保存機能

### Word_しおり整理ツール/
Word文書のしおり（ブックマーク）整理とPDF出力ツールです。

- ExcelからWordを操作し、スタイルに基づいてアウトラインレベルを自動設定
- 「表題1」「表題2」「表題3」などの独自スタイルに対応
- しおり付きPDFを自動出力
- Input/Output方式でファイル管理が明確

### Git_Log_可視化ツール/
Gitコミット履歴の可視化ツールです。

- Excelから実行し、git logを表形式・統計・グラフで視覚化
- 5つのシート：Dashboard、CommitHistory、Statistics、Charts、BranchGraph
- 作者別・日別の統計情報を自動集計
- 全ブランチ対応で最近のN件を取得
- ブランチ構造の図形可視化機能

### Excel_Word_ファイル比較ツール/
Excel/Wordファイル比較ツールです。

- 2つのExcelファイルを比較し、差異を一覧表示
- 2つのWordファイルを比較（段落単位/詳細比較）
- シート単位・セル単位での差異検出
- 差異の種類を識別（値変更、追加、削除）
- 結果をExcelシートに出力
- 差異セルのハイライト表示

### Git_コマンド解説書/
Git初心者向けのコマンド解説書を生成するツールです。

- 8つのシートに分かれた包括的なGit解説書を自動生成
- 基礎知識、基本コマンド、ブランチ操作、リモート操作等を網羅
- 実践シナリオやトラブル対処も収録
- 初心者向けの用語解説付き
- 研修資料やリファレンスとして活用可能

### JP1_ジョブ管理ツール/
JP1/AJS3のジョブネット一覧取得・一括実行ツールです。

- ローカル/リモート実行モード選択可能
- ajsprintでジョブネット一覧を取得してExcelに表示
- チェック＋実行順指定で選択ジョブを順次実行
- 完了待ち機能（正常終了/異常終了を判定）
- 1つでも失敗したら処理を中断
- WinRM設定の自動管理・復元（リモートモード時）
- 実行ログの自動記録

### JP1_ジョブ管理ツール_バッチ版/
JP1/AJS3のジョブ管理ツール（外部バッチファイル版）です。

- ジョブ実行処理を外部バッチファイルに分離
- 設定をバッチファイル内で管理（空欄時はパスワード入力プロンプト）
- バッチファイル単体でもスタンドアロン実行可能
- 保留ジョブの自動解除・ハイライト表示対応
- 通常版と同等の機能（完了待ち、エラー停止、ログ出力）
- Excel VBAとバッチファイルの2ファイル構成

### PDF_しおり検証ツール/
PDFのしおり（ブックマーク）検証ツールです。

- PDFからすべてのしおり（階層構造含む）を取得
- 各しおりのリンク先ページ番号を表示
- しおり名がリンク先ページに存在するかチェック
- 一致率によるOK/NG/確認要の判定を自動表示
- 検証結果をCSVファイルに出力可能
- iTextSharp（.NET）を使用、IT制限環境対応

## 使い方

1. 各サブフォルダ内のREADME.mdを参照
2. .xlsmファイルまたは.basファイルをダウンロード
3. Excelで開くか、VBAエディタでインポート
4. マクロを実行

## 注意事項

- マクロを有効にする必要があります
- Excel 2010以降を推奨
- ファイルは必ずマクロ有効ブック(.xlsm)として保存してください
