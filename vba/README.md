# VBA Tools

Excel VBAマクロによる自動化ツール集です。

## 📂 サブフォルダ

### excel-automation/
Excel業務を自動化するVBAマクロを格納します。

- 勤怠管理の自動化
- データ集計・レポート生成
- フォーム入力の自動化
- その他Excel関連の繰り返し作業の効率化

## 使い方

1. 各サブフォルダ内のREADME.mdを参照
2. .xlsmファイルまたは.basファイルをダウンロード
3. Excelで開くか、VBAエディタでインポート
4. マクロを実行

## 注意事項

- マクロを有効にする必要があります
- Excel 2010以降を推奨
- ファイルは必ずマクロ有効ブック(.xlsm)として保存してください
