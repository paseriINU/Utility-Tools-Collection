# Git コマンド解説書 生成ツール

Git初心者向けのコマンド解説書をExcelで生成するVBAツールです。

## 概要

このツールを実行すると、Gitの基礎知識から実践的な使い方まで、8つのシートに分かれた包括的な解説書が生成されます。

## 機能

### 生成されるシート

| シート名 | 内容 |
|---------|------|
| **Git基礎知識** | Gitの概念、用語説明、3つの領域の解説 |
| **基本コマンド** | init, clone, status, add, commit, push, pull等 |
| **ブランチ操作** | branch, checkout, switch, merge等 |
| **リモート操作** | remote, push, pull, fetchの違い等 |
| **履歴・差分確認** | log, diff, show, blame等 |
| **取り消し・修正** | reset, revert, checkout, amend等 |
| **実践シナリオ** | 新規プロジェクト開始、機能開発等の実例 |
| **トラブル対処** | よくある問題とその解決方法 |

### 特徴

- **初心者向け**: 専門用語には読み方と説明を記載
- **実践的**: 実際の作業フローをステップで解説
- **見やすい**: 色分け、表形式で整理
- **すぐ使える**: コピー&ペーストできるコマンド例

## 必要な環境

- Microsoft Excel 2010以降

## 使い方

### インストール

1. Excelを開く
2. `Alt + F11` でVBAエディタを開く
3. 「挿入」→「標準モジュール」
4. `GitCommandGuide.bas` の内容をコピー&ペースト
5. VBAエディタを閉じる

### 実行

1. `Alt + F8` でマクロダイアログを開く
2. `CreateGitCommandGuide` を選択して「実行」
3. 8つのシートが自動生成されます

## 解説書の内容

### Git基礎知識
- Gitとは何か
- なぜGitを使うのか
- 重要な用語（リポジトリ、コミット、ブランチ等）
- 3つの領域（作業ディレクトリ、ステージング、リポジトリ）

### 基本コマンド
```
git status    # 状態確認（迷ったらこれ!）
git add .     # 全ファイルをステージング
git commit -m "メッセージ"  # 変更を記録
git push      # リモートに送信
git pull      # リモートから取得
```

### ブランチ操作
```
git branch              # ブランチ一覧
git checkout -b 名前    # ブランチ作成&切替
git merge 名前          # ブランチを統合
```

### トラブル対処
- pushできない時の対処
- コンフリクト（競合）の解決方法
- 間違えたコミットの修正
- 変更を消してしまった時の復旧

## 活用方法

2. **デスク常備**: コマンドを忘れた時の参照用
3. **チーム共有**: プロジェクトメンバーへの配布
