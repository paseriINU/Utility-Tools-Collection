# 便利ツール集 (Utility Tools Collection)

開発効率化・業務自動化のための便利ツールを言語・用途別に整理したリポジトリです。

## 🛠️ 現在利用可能なツール

### Batch Scripts

**Note**: このカテゴリにはWindowsバッチファイル(.bat)およびPowerShellスクリプト(.ps1)が含まれます。
- **[TFS to Git 同期ツール](batch/TFS_to_Git_同期ツール/)**: TFS（Team Foundation Server）とGitリポジトリを同期するバッチスクリプト
  - MD5ハッシュによる高速差分チェック
  - 自動的なファイル更新・追加・削除
  - 日本語ファイル名対応

- **[リモートバッチ実行ツール](batch/リモートバッチ実行ツール/)**: リモートWindowsサーバでバッチファイルを実行（PowerShell Remoting）
  - ダブルクリックで実行可能な.batハイブリッドスクリプト
  - WinRM設定の自動構成と復元（TrustedHosts自動設定）
  - 環境選択機能（tst1t/tst2t）
  - 実行結果のリアルタイム表示と終了コード取得
  - ログファイル自動保存
  - ネットワークパス（UNCパス）対応

- **[Git 差分ファイル抽出ツール](batch/Git_差分ファイル抽出ツール/)**: Gitブランチ間の差分ファイルを抽出
  - フォルダ構造を保ったまま差分ファイルをコピー
  - main と develop などブランチ間の差分を簡単に抽出
  - ネットワークフォルダへの自動出力機能（ブランチ名照合）
  - デプロイ用差分ファイル作成に最適
  - ダブルクリックで実行可能

- **[Git ブランチ削除ツール](batch/Git_ブランチ削除ツール/)**: Gitブランチを数字で選択して削除
  - リモートブランチを対話的に削除
  - ローカルブランチを対話的に削除
  - リモート＆ローカル両方を一度に削除
  - main/master は保護機能付き（選択不可）
  - 通常削除・強制削除を選択可能

- **[Git ブランチ上書きツール](batch/Git_ブランチ上書きツール/)**: Gitブランチを別のブランチで完全上書き
  - git reset --hard 方式で完全置換
  - バックアップブランチの自動作成機能
  - main/masterブランチは保護（選択不可）
  - リモートへの --force-with-lease 対応
  - ダブルクリックで実行可能

- **[Git Linuxデプロイツール](batch/Git_Linuxデプロイツール/)**: Git変更ファイルをLinuxサーバーに転送
  - 複数環境対応・拡張子フィルタ・削除ファイル除外
  - すべて転送 or 個別選択の2つのモード
  - クリーンアップ機能（転送先フォルダを事前削除してからコピー）
  - Linux側の自動設定（mkdir/chmod/chown）
  - SCP/SSH対応（Windows OpenSSH Client）
  - ネットワークパス（UNCパス）からの実行対応
  - ダブルクリックで実行可能

- **[JP1 ジョブツール](batch/JP1_ジョブツール/)**: JP1/AJS3 Web Console REST APIでジョブ実行・ログ取得
  - 2つのメインツール: 即時実行+ログ取得 / ログ取得のみ（比較モード対応）
  - サンプルバッチ付き（EXEC/GET切替、Excel貼り付け対応）
  - WinRM不要（Web Console経由）
  - ダブルクリックで実行可能

- **[サーバ構成情報収集ツール](batch/サーバ構成情報収集ツール/)**: サーバ構成情報収集ツール
  - ネットワーク・セキュリティ設定をExcel出力
  - WinRM設定、ファイアウォール、開放ポート、レジストリ
  - WinRM実行前の事前調査に最適

- **[Git 開発環境初期設定ツール](batch/Git_開発環境初期設定ツール/)**: Git開発環境のセットアップを一括実行
  - Git globalのuser.name/user.email自動設定
  - SSH鍵ペア（Ed25519/RSA）の生成
  - Linuxサーバーへの公開鍵自動配置（重複チェック付き）
  - 接続テストの自動実行
  - Gitリポジトリのクローン（オプション）
  - ダブルクリックで実行可能

- **[ファイル拡張子解析ツール](batch/ファイル拡張子解析ツール/)**: フォルダ内の拡張子を一覧表示
  - サブフォルダを含めて再帰的に検索
  - 拡張子ごとのファイル数と割合を表示
  - フォルダ別に拡張子を表示する機能
  - 拡張子別ファイル一覧表示機能
  - CSV形式でのエクスポート機能
  - ダブルクリックで実行可能

- **[VB.NET ソース改修デプロイツール](batch/VBNet_ソース改修デプロイツール/)**: VB.NETソースの改修・デプロイ・ビルドを自動化
  - 改修済みソースフォルダの構成を自動取得
  - メインフォルダから対象ファイルを日付付きフォルダにバックアップ
  - 改修済みファイルをメインフォルダに上書きコピー
  - MSBuildでReleaseビルド（Visual Studio自動検出）
  - 生成されたDLLをデスクトップに自動コピー
  - ダブルクリックで実行可能

### Linux Tools

- **[WinRM-Client](linux/winrm-client/)**: LinuxからWindowsへWinRM接続してコマンド実行
  - Python版・Bash版・C言語版の3種類を提供
  - 追加パッケージ不要（標準ライブラリのみ）
  - IT制限環境対応

- **[OpenTP1 デプロイツール](linux/opentp1-deploy/)**: OpenTP1環境でのソース配置を自動化
  - OpenTP1停止 → ソース配置 → 起動を自動実行
  - バックアップ機能付き
  - 詳細なログ出力

- **[C言語 メモリ安全性チェックツール](linux/c-memory-safety-check/)**: C言語のメモリ安全性・デッドロジックを検出
  - GCC警告オプションによる静的チェック
  - cppcheckによる静的解析（オプション）
  - AddressSanitizerによるメモリ破壊検出
  - Valgrindによるメモリリーク検出（オプション）
  - 到達不能コード・未使用変数の検出

### Git Hooks

- **[Git-Hooks](git-hooks/)**: リモートリポジトリ保護設定
  - master/mainブランチへの直接プッシュ防止
  - 機密情報・大きなファイルの検出
  - コミットメッセージ品質チェック

### VBA Macros

- **[SQL 生成ツール](vba/SQL_生成ツール/)**: Oracle SELECT文を対話的に生成するツール
  - Excel上でGUIを使ってSQL文を組み立て
  - JOIN（INNER, LEFT, RIGHT, FULL OUTER, CROSS）対応
  - サブクエリ、WITH句（CTE）、UNION対応
  - 集計関数（COUNT, SUM, AVG, MAX, MIN）、GROUP BY / HAVING対応
  - SQLヘルプシートでSQL初心者も安心して使用可能
  - 生成履歴の保存機能

- **[Word しおり整理ツール](vba/Word_しおり整理ツール/)**: Word文書のしおり（ブックマーク）整理とPDF出力
  - ExcelからWordを操作し、スタイルに基づいてアウトラインレベルを自動設定
  - 「表題1」「表題2」「表題3」などの独自スタイルに対応
  - しおり付きPDFを自動出力（目次で適切な箇所に飛べる）
  - Input/Output方式でファイル管理が明確
  - スタイル名のカスタマイズ可能

- **[Git Log 可視化ツール](vba/Git_Log_可視化ツール/)**: Gitコミット履歴の可視化ツール
  - Excelから実行し、git logを表形式で視覚化
  - ダッシュボードでサマリー情報・作者別統計を表示
  - メインシートで設定、コミット履歴シートに結果出力
  - 全ブランチ対応で最近のN件を取得
  - コミットメッセージ全文・変更行数も表示
  - 環境変数（%USERNAME%等）をパス内で使用可能

- **[Excel/Word ファイル比較ツール](vba/Excel_Word_ファイル比較ツール/)**: Excel/Wordファイル比較ツール
  - 2つのExcelファイルを比較し、差異を一覧表示
  - 2つのWordファイルを比較（WinMerge方式・LCSアルゴリズム）
  - シート単位・セル単位での差異検出
  - 差異の種類を識別（値変更、追加、削除、スタイル変更）
  - 1行追加/削除があっても以降の行がすべて差異にならない
  - Word差分の検索ジャンプ機能（差分テキストを検索して該当箇所に自動ジャンプ）
  - 差異セルのハイライト表示（黄色:変更、緑:追加、赤:削除）

- **[Git コマンド解説書](vba/Git_コマンド解説書/)**: Git初心者向けコマンド解説書を生成
  - 8つのシートに分かれた包括的なGit解説書を自動生成
  - 基礎知識、基本コマンド、ブランチ操作、リモート操作を網羅
  - 実践シナリオ（新規プロジェクト、機能開発等）の手順解説
  - トラブル対処法（コンフリクト解決、間違いの修正等）
  - 初心者向けの用語解説・読み方付き
  - 研修資料やリファレンスとして活用可能

- **[JP1 ジョブ管理ツール](vba/JP1_ジョブ管理ツール/)**: JP1ジョブネット一覧取得・一括実行
  - ローカル/リモート実行モード選択可能
  - ajsprintでジョブネット一覧を取得してExcelに表示
  - チェック＋実行順指定で選択ジョブを順次実行
  - 完了待ち機能（正常終了/異常終了を判定）
  - 1つでも失敗したら処理を中断
  - 実行ログの自動記録

- **[JP1 ジョブ管理ツール（バッチ版）](vba/JP1_ジョブ管理ツール_バッチ版/)**: JP1ジョブ管理（外部バッチファイル版）
  - ジョブ実行処理を外部バッチファイルに分離
  - 設定をバッチファイル内で管理（空欄時はパスワード入力プロンプト）
  - バッチファイル単体でもスタンドアロン実行可能
  - 保留ジョブの自動解除・ハイライト表示対応
  - 通常版と同等の機能（完了待ち、エラー停止、ログ出力）

- **[JP1 REST ジョブ管理ツール](vba/JP1_REST_ジョブ管理ツール/)**: JP1 REST APIでジョブ管理（WinRM不使用）
  - JP1/AJS3 Web Console REST APIを使用（WinRM不要）
  - ツリー表示でジョブ階層を展開/折りたたみ
  - ジョブネットの即時実行登録
  - 実行結果詳細（ログ）の取得
  - 完了待ち機能で自動ログ取得

### JavaScript Tools

- **[PDF 比較ツール](javascript/PDF_比較ツール/)**: 2つのPDFファイルをビジュアル比較するブラウザベースのツール
  - 完全スタンドアローン（1つのHTMLファイルで完結、ライブラリ埋め込み済み）
  - 追加インストール不要・インターネット接続不要
  - 全ページを縦に並べてスクロールで閲覧
  - 左右並列表示で差分を色付きハイライト（黄=変更、緑=追加、赤=削除）
  - ドラッグ&ドロップ対応、同期スクロール機能

- **[PDF テキスト抽出ツール](javascript/PDF_テキスト抽出ツール/)**: PDFからテキストを抽出してWinMergeで比較するツール
  - PDF.jsでテキスト抽出、BOM付きUTF-8でダウンロード
  - 画像比較より正確なテキストベース差分検出
  - 2つのPDFを読み込み、テキストファイルとして出力
  - WinMergeで文字内容の差分を確認可能
  - ドラッグ&ドロップ対応、プレビュー表示

- **[PDF しおり検証ツール](javascript/PDF_しおり検証ツール/)**: PDFのしおり（ブックマーク）を検証するブラウザベースのツール
  - 完全スタンドアローン（1つのHTMLファイルで完結、外部ライブラリ不要）
  - PDFからすべてのしおり（階層構造含む）を取得
  - 各しおりのリンク先ページ番号を表示
  - OK/確認要の判定を自動表示
  - 検証結果をCSVファイルに出力可能
  - クロスプラットフォーム対応（Windows/Mac/Linux）

4. このREADME.mdの「現在利用可能なツール」セクションを更新
