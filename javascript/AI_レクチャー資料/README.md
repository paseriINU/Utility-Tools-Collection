## 使用方法

### スライドの開き方

1. `Claude_Code_実践ガイド.html` をブラウザで開く
2. スライドショーが自動的に開始されます

### キーボード操作

| キー | 操作 |
|------|------|
| `→` / `Space` | 次のスライドへ |
| `←` | 前のスライドへ |
| `Esc` / `O` | スライド一覧表示 |
| `F` | フルスクリーンモード |
| `S` | スピーカーノート表示 |
| `B` / `.` | 画面を暗転 |
| `?` | ヘルプ表示 |

### スライド番号で移動

URLの末尾に `#/スライド番号` を追加すると、特定のスライドに直接移動できます。

例: `Claude_Code_実践ガイド.html#/10`

## カスタマイズ

### スライドの編集

HTMLファイルを直接編集することで、内容をカスタマイズできます。

```html
<section>
    <h2>スライドタイトル</h2>
    <p>コンテンツ</p>
</section>
```

### テーマの変更

`<head>` 内のCSSファイルを変更することで、テーマを切り替えられます。

利用可能なテーマ:
- `night.css` (デフォルト・ダーク)
- `white.css` (ライト)
- `black.css` (シンプルダーク)
- `league.css` (グレー)
- `beige.css` (ベージュ)

## 動作環境

- **推奨ブラウザ**: Chrome, Firefox, Safari, Edge（最新版）
- **インターネット接続**: reveal.jsをCDNから読み込むため必要

### オフライン対応

オフライン環境で使用する場合は、reveal.jsをローカルにダウンロードしてパスを変更してください。
