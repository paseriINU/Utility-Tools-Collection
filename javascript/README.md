# JavaScript Tools

JavaScriptで作成したブラウザベースのツール集です。

## ツール一覧

### PDF_比較ツール

2つのPDFファイルを視覚的に比較するブラウザベースのツールです。

**機能:**
- PDFを左右に並べて表示、差分をカラーハイライト
- ドラッグ&ドロップでファイル読み込み
- 同期スクロール、ズーム機能
- 追加インストール不要（HTMLファイルをダブルクリックで起動）
- オフライン対応（ローカル優先・CDNフォールバック方式）

**使い方:**
1. `PDF_比較ツール/pdf_visual_compare.html` をブラウザで開く
2. 左右のパネルに2つのPDFをドラッグ&ドロップ
3. 自動で比較開始、差分が色付きで表示される

詳細は [PDF_比較ツール/README.md](PDF_比較ツール/README.md) を参照

### PDF_しおり検証ツール

PDFのしおり（ブックマーク）を検証するブラウザベースのツールです。

**機能:**
- PDFからすべてのしおり（階層構造含む）を取得
- 各しおりのリンク先ページ番号を表示
- OK/確認要の判定を自動表示
- 検証結果をCSVファイルに出力
- 外部ライブラリ不要（標準Web APIのみ使用）
- オフライン対応

**使い方:**
1. `PDF_しおり検証ツール/PDF_しおり検証ツール.html` をブラウザで開く
2. PDFファイルをドラッグ&ドロップ
3. 「しおり検証」ボタンをクリック
4. 結果をテーブルで確認、必要に応じてCSV出力

詳細は [PDF_しおり検証ツール/README.md](PDF_しおり検証ツール/README.md) を参照

## 実行方法

### HTMLファイルを直接開く（推奨）
1. HTMLファイルをダブルクリック
2. ブラウザで自動的に開きます

### ブラウザコンソールで実行
1. ブラウザの開発者ツールを開く（F12）
2. コンソールタブを選択
3. スクリプトをコピー＆ペースト
4. Enterキーで実行

### ブックマークレットとして使用
1. スクリプトをブックマークのURLフィールドに登録
2. 対象のWebページでブックマークをクリック

## 注意事項

- 一部のツールはインターネット接続が必要です（CDNからライブラリを読み込むため）
- スクリプトは対象のWebサイトの構造に依存する場合があります
- Webサイトの更新により動作しなくなる可能性があります
- 自己責任で使用してください
