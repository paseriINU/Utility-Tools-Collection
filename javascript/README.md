# JavaScript Tools

JavaScriptで作成したブラウザベースのツール集です。

## ツール一覧

### PDF_比較ツール

2つのPDFファイルを比較するブラウザベースのツールです。2種類の比較方式を提供しています。

| ファイル | 比較方式 | 用途 |
|---------|---------|------|
| `pdf_compare.html` | テキスト比較 | テキストベースPDFの内容比較、CSV出力 |
| `pdf_visual_compare.html` | 視覚比較 | レイアウト・画像含む見た目の比較 |

**機能:**
- **テキスト比較**: LCSアルゴリズムによる高精度な差分検出、WinMerge方式表示、CSVエクスポート
- **視覚比較**: PDFを左右に並べて表示、差分をカラーハイライト、同期スクロール、ズーム機能
- 追加インストール不要（HTMLファイルをダブルクリックで起動）
- オフライン対応（ローカル優先・CDNフォールバック方式）

**使い方:**
1. `PDF_比較ツール/pdf_compare.html` または `pdf_visual_compare.html` をブラウザで開く
2. 2つのPDFをドラッグ&ドロップ
3. テキスト比較: 「比較を実行」をクリック / 視覚比較: 自動で比較開始

詳細は [PDF_比較ツール/README.md](PDF_比較ツール/README.md) を参照

## 実行方法

### HTMLファイルを直接開く（推奨）
1. HTMLファイルをダブルクリック
2. ブラウザで自動的に開きます

### ブラウザコンソールで実行
1. ブラウザの開発者ツールを開く（F12）
2. コンソールタブを選択
3. スクリプトをコピー＆ペースト
4. Enterキーで実行

### ブックマークレットとして使用
1. スクリプトをブックマークのURLフィールドに登録
2. 対象のWebページでブックマークをクリック

## 注意事項

- 一部のツールはインターネット接続が必要です（CDNからライブラリを読み込むため）
- スクリプトは対象のWebサイトの構造に依存する場合があります
- Webサイトの更新により動作しなくなる可能性があります
- 自己責任で使用してください
