# JavaScript Tools

JavaScriptで作成したブラウザベースのツール集です。

## ツール一覧

### PDF_比較ツール

2つのPDFファイルを比較し、差異を一覧表示するブラウザベースのツールです。

**機能:**
- LCS（最長共通部分列）アルゴリズムによる高精度な差分検出
- WinMerge方式の表示（旧/新行番号を両方表示）
- 追加インストール不要（HTMLファイルをダブルクリックで起動）
- オフライン対応（ローカル優先・CDNフォールバック方式）
- CSVエクスポート機能

**使い方:**
1. `PDF_比較ツール/pdf_compare.html` をブラウザで開く
2. 2つのPDFをドラッグ&ドロップ
3. 「比較を実行」をクリック

詳細は [PDF_比較ツール/README.md](PDF_比較ツール/README.md) を参照

## 実行方法

### HTMLファイルを直接開く（推奨）
1. HTMLファイルをダブルクリック
2. ブラウザで自動的に開きます

### ブラウザコンソールで実行
1. ブラウザの開発者ツールを開く（F12）
2. コンソールタブを選択
3. スクリプトをコピー＆ペースト
4. Enterキーで実行

### ブックマークレットとして使用
1. スクリプトをブックマークのURLフィールドに登録
2. 対象のWebページでブックマークをクリック

## 注意事項

- 一部のツールはインターネット接続が必要です（CDNからライブラリを読み込むため）
- スクリプトは対象のWebサイトの構造に依存する場合があります
- Webサイトの更新により動作しなくなる可能性があります
- 自己責任で使用してください
