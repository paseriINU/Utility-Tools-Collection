# JavaScript Tools

JavaScriptで作成したブラウザ自動化・Web関連ツール集です。

## 📂 ツール一覧

*現在、ツールは準備中です。今後追加予定。*

追加予定のツール例:
- Webフォーム自動入力
- ブックマークレット
- ブラウザコンソールで実行するスクリプト
- 特定のWebサイト向けの自動化ツール

## 実行方法

### ブラウザコンソールで実行
1. ブラウザの開発者ツールを開く（F12）
2. コンソールタブを選択
3. スクリプトをコピー＆ペースト
4. Enterキーで実行

### ブックマークレットとして使用
1. スクリプトをブックマークのURLフィールドに登録
2. 対象のWebページでブックマークをクリック

## 注意事項

- スクリプトは対象のWebサイトの構造に依存します
- Webサイトの更新により動作しなくなる可能性があります
- 自己責任で使用してください
- 利用規約に違反しないよう注意してください
