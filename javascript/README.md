# JavaScript Tools

JavaScriptで作成したブラウザ自動化・Web関連ツール集です。

## 📂 サブフォルダ

### browser-automation/
ブラウザ上で動作する自動化スクリプトを格納します。

- Webフォーム自動入力
- ブックマークレット
- ブラウザコンソールで実行するスクリプト
- 特定のWebサイト向けの自動化ツール

## 実行方法

### ブラウザコンソールで実行
1. ブラウザの開発者ツールを開く（F12）
2. コンソールタブを選択
3. スクリプトをコピー＆ペースト
4. Enterキーで実行

### ブックマークレットとして使用
1. スクリプトをブックマークのURLフィールドに登録
2. 対象のWebページでブックマークをクリック

## 注意事項

- スクリプトは対象のWebサイトの構造に依存します
- Webサイトの更新により動作しなくなる可能性があります
- 自己責任で使用してください
- 利用規約に違反しないよう注意してください

## 開発環境

Node.js環境での開発も可能です:

```bash
npm install
npm run dev
```
