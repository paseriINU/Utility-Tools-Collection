# Python Utilities

Python で作成した汎用ユーティリティツール集です。

## 📂 サブフォルダ

### utilities/
汎用的なPythonスクリプトを格納します。

- データ処理・変換ツール
- ファイル操作の自動化
- API連携スクリプト
- 機械学習の実験コード
- 画像処理ツール

## 必要な環境

- Python 3.8以降推奨
- 各スクリプトで必要なライブラリは個別のREADME.mdに記載

## 使い方

1. 各サブフォルダ内のREADME.mdを参照
2. 必要に応じて依存ライブラリをインストール
   ```bash
   pip install -r requirements.txt
   ```
3. スクリプトを実行
   ```bash
   python script_name.py
   ```

## 開発環境

仮想環境の使用を推奨します:

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```
